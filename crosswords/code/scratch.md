# Find all words that match on all permutations of a certain set of letters

Generate a Python script that performs the following:

- Reads in a list of words from a file (delimited by newline).
- Finds all words from that file matching a set of constraints.

Constraints:

- The word is exactly _n_ letters long.
- The word must have the letter R at position 8.
- The word must have the letters O and S at positions 4 and 12, in either order, but not both.
- The word must have the letters E and N at positions 6 and 10, in either order, but not both.
